📌 Project Overview
This project focuses on analyzing the Amazon Sale Report dataset to understand sales performance, customer trends, and product-level insights. The dataset was sourced from Kaggle and processed using Python in Google Colab.

🎯 Objectives
Import and handle real-world e-commerce data
Perform data cleaning and preprocessing
Explore dataset structure and data types
Prepare data for exploratory data analysis (EDA)
Generate insights for business decision-making

🛠️ Tools & Technologies
🐍 Python
📊 Pandas
🔢 NumPy
☁️ Google Colab

📂 Dataset Information
Dataset Name: Amazon Sale Report
Source: Kaggle
Format: CSV (Zipped file)

⚙️ Workflow
 1. Data Collection
Downloaded dataset from Kaggle
Uploaded ZIP file into Google Colab
 2. Data Extraction
Extracted dataset using zipfile library
 3. Data Loading
Loaded CSV file using Pandas
 4. Data Exploration
Viewed dataset using .head()
Checked data types using .dtypes
Understood dataset structure
 5. Data Preprocessing
Checked for missing values
Cleaned and prepared dataset


📈 Key Learnings
Handling ZIP datasets in Python
Working with real-world e-commerce data
Data cleaning and preprocessing
Building foundation for EDA

🚀 Future Scope
Perform advanced EDA
